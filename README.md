# icecream-shop-website
"Sweet Treats" is a comprehensive e-commerce web application designed for an online ice cream parlor. The project focuses on providing a seamless user experience for browsing and ordering frozen desserts.
✨ Key Features
Product Catalog: A dynamic display of various ice cream flavors and treats.

Shopping Cart: An interactive system that allows users to add, remove, and manage their selections before checkout.

Order Management: A backend structure to handle customer orders and processing.

Responsive Design: Optimized for both desktop and mobile users to ensure easy navigation.

🛠️ Tech Stack
Frontend: Developed using HTML, CSS, and JavaScript for an interactive UI.

Backend: Built with PHP to handle server-side logic and requests.

Database: Utilizes MySQL and MongoDB for efficient data storage and management.

Development Environment: Managed and tested using XAMPP
